# 小墨个人漫剧工作流 Skills

这个仓库包含小墨个人的 AI 漫剧生产 Skills，可以直接在 Grok 中使用。

## 已安装的 Skills（本地已部署）

### 1. storyboard-director（电影分镜导演）
- 触发词：分镜、storyboard、文字分镜、电影分镜、生成分镜、帮我分镜、导演分镜、镜头脚本
- 功能：把小说/剧本原文转成专业电影文字分镜，直接喂给 Runway / Sora / Kling 等 AI 视频工具
- 强制约束：自然光优先、黄金时刻、统一色调、景别运镜丰富、每镜含英文 AI 提示词、无 BGM

### 2. character-analyzer（角色道具提取）
- 触发词：分析人物、分析角色、人物外观分析、角色视觉分析、提取人物设定
- 功能：从小说/剧本提取角色外观描述（一整段文字）+ 独立道具设计清单

## 其他文件
- `漫剧工作流.docx`：完整 AI 漫剧创作工作流（小说写作节奏 + 3D/仿真人漫角色场景道具提示词提取 + 分镜工作流）
- `分镜提示词格式.txt`：分镜输出示例格式
- `场景人物道具资产提示词.txt`：人物四视图、道具四宫格、场景资产提示词模板

## 使用方法
在 Grok 对话中直接说触发词即可激活对应 Skill。
例如：
- 「帮我分镜这段小说」
- 「分析一下这个角色的外观」

本地 Skills 已部署到当前环境，新会话也会生效。
